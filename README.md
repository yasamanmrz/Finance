# Finance
Personal Finance Tracker

A modern and responsive personal finance tracker built with HTML, Tailwind CSS, and Vanilla JavaScript. This application helps users manage their income and expenses, monitor their financial activity, and maintain better control over their budget.

Features

- Add and manage income and expense transactions
- Real-time balance calculation
- Transaction history tracking
- Responsive design optimized for mobile, tablet, and desktop
- Clean and user-friendly interface
- Local storage support for persistent data
- Fast and lightweight performance

Technologies Used

- HTML5
- Tailwind CSS
- JavaScript (ES6)

Learning Objectives

This project was developed to strengthen front-end development skills, including:

- DOM manipulation
- Event handling
- Data management with Local Storage
- Responsive web design
- Building interactive user interfaces with Tailwind CSS
